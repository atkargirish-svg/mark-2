
# REVO Console - Next-Gen Robotic Arm Control

This is the professional control interface for the **REVO Mark II** 4-Axis Robotic Arm.

## Core Features
- **REVO Brain AI**: High-level command processing via JARVIS personality.
- **Precision Manual Link**: Real-time 4-axis control with hardware inversion logic.
- **Glassmorphism UI**: Futuristic, high-performance web console.
- **Firebase RTDB**: Ultra-low latency communication with ESP8266/NodeMCU hardware.
